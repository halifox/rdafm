# 📻 RDA FM Radio 示例程序（Android）

这是一个基于 Android 平台的 RDA FM Radio 示例应用，适用于集成了 **RDA5807** 或兼容芯片的设备。该程序使用 JNI 或直接访问底层接口，实现 FM 调频广播的控制与播放。

---

## ✨ 功能亮点

- ✅ 打开 / 关闭 FM 模块
- 🎚️ 调频
- 🔍 扫描电台
- 🔊 音量控制
- ⭐ 收藏喜爱频道

---

## 🛠️ 环境要求

- 📱 Android 6.0 及以上
- 🔓 已 root 或具备访问 `/dev/RDAFM` 等设备节点的权限
- 📡 搭载 RDA5807 或兼容 FM 接收芯片
- 🧩 系统支持 JNI / HAL 层访问
